# Tax + Tip Calculator
## What it does
Calculates tax, tip, and per-person split from a subtotal with configurable tax and tip percentages.
## Endpoints
| Path | Method | Description |
|------|--------|-------------|
| `GET /` | GET | Tax + tip calculator |
| `GET /widget.js` | GET | Iframe injector |
## Embed
```html
<script src="https://your-worker.workers.dev/widget.js"></script>
```
## Tips
- Stateless – no bindings required.
