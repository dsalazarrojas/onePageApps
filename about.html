About One-Page Apps

A curated library of 200 free, single-file web apps you can try instantly in your browser and deploy to Cloudflare Workers in one click.

What is this?

onePageApps.gic.mx is a showcase and distribution site for 200 ready-to-deploy Cloudflare Worker scripts — everything from AI text tools and image editors to calculators, converters, and embeddable widgets.

Each app is a single .js file you can copy, customise, and deploy to the Cloudflare Workers free tier in seconds. No build step, no dependencies, no server to manage.

The collection is also available through the One-Page Apps iOS and macOS app, which lets you browse, preview, and deploy apps directly from your device.

How it works

  1. 1

    Browse or search

    Use the Browse page or category pages to find an app that fits your needs.

  2. 2

    Try it live

    Each app has a live preview page so you can test it right here in your browser before deploying anything.

  3. 3

    Deploy to Cloudflare Workers

    Copy the script (or use the iOS app to deploy with one tap). The Cloudflare free tier covers most personal and small-project usage.
